# AHT20 + BMP280 模块面包板接线图

> 说明：AHT20 与 BMP280 为 I2C 复用模块，两个传感器共用同一 I2C 总线（SDA/SCL）。

## 接线示意（ASCII）

```
ESP32-S3 DevKitC-1            AHT20 + BMP280 模块
┌───────────────┐             ┌──────────────────┐
│ 3V3       GND │────────────▶│ VCC         GND  │
│ GPIO8   GPIO9 │             │ SDA         SCL  │
└───────────────┘             └──────────────────┘
       │        │
       └────────┴── I2C 总线复用 (SDA=GPIO8, SCL=GPIO9)
```

## 详细连线表

| ESP32-S3 引脚 | 连接到模块 | 说明 |
| --- | --- | --- |
| 3V3 | VCC | 3.3V 供电 |
| GND | GND | 地 |
| GPIO8 (SDA) | SDA | I2C 数据线 |
| GPIO9 (SCL) | SCL | I2C 时钟线 |

## 备注

- 模块通常内置 I2C 上拉电阻；若模块未带上拉，需在 SDA/SCL 上拉 4.7kΩ 到 3.3V。
- 若与其他 I2C 设备共线，确保地址区分：AHT20 = 0x38，BMP280 = 0x76（或 0x77）。
